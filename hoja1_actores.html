Hoja 1 — Identificación de Actores

Sistema Analizado: Plataforma de Banca Digital

Este documento forma parte del análisis del sistema Plataforma de Banca Digital, desarrollado para permitir a los clientes de un banco gestionar sus cuentas, realizar transferencias y pagar servicios sin necesidad de acudir físicamente a una sucursal bancaria. Durante esta fase se identifican las entidades externas que interactúan con el sistema para posteriormente construir los casos de uso y el diagrama UML correspondiente.
Definición UML: Un actor representa cualquier entidad externa al sistema que interactúa con él. Puede ser una persona, organización o sistema externo. Los actores pueden clasificarse como primarios, cuando inician la interacción con el sistema, o secundarios/exteriores, cuando colaboran en la ejecución de procesos.
Actores identificados
👤
Actor Primario
Cliente
Usuario del banco que utiliza la aplicación digital para realizar operaciones financieras de manera remota desde dispositivos móviles o computadoras.
  • Iniciar sesión en la plataforma
  • Consultar saldo y movimientos
  • Realizar transferencias bancarias
  • Pagar servicios públicos
🛡️
Actor Secundario
Administrador del Sistema
Empleado autorizado del banco encargado de administrar la plataforma y supervisar las operaciones realizadas dentro del sistema.
  • Gestionar la información de usuarios
  • Bloquear o habilitar cuentas
  • Supervisar operaciones realizadas
  • Garantizar la seguridad del sistema
🏢
Actor Externo
Empresa Proveedora de Servicios
Entidad externa que interactúa con el sistema bancario para validar y procesar pagos de servicios públicos como electricidad, agua, telefonía o internet.
  • Validar información del servicio
  • Confirmar pagos realizados
  • Procesar transacciones de servicios
Resumen de actores
# Actor Tipo Rol en el sistema Interacción
1 Cliente Primario Usuario que utiliza la aplicación bancaria Directa
2 Administrador Secundario Gestiona y supervisa el sistema Directa
3 Proveedor de Servicios Externo Procesa pagos de servicios públicos Indirecta
